# Creative Image Card Effects

Created 3 Card hover effects for people to freely use.

## Description

We all visit many sites and we all have seen those cool visual cards with cool hover effects on them. So I decided to create the 3 card hover effects for people to use for their own portfolio or their website.

[GitHub page](https://github.com/Digita1Panda/creative-image-card-effects)

## Installation

- Copy the relevant part of the HTML and CSS code into your own and replace the image to your choosing.

## Usage

- Image Card:

  ```
  Hover over the image. Make sure to put your own image in the <img src=""> tag.
  ```
